# Llama-Factory

LLaMA-Factory is a framework for fine-tuning LLMs. It provides a no-code WebUI,
supports 100+ models, parameter-efficient methods (LoRA/QLoRA), and multimodal
training.

## When to use

- Fine-tuning LLMs without writing training code (WebUI)
- LoRA/QLoRA fine-tuning across many model families
- Multimodal fine-tuning
